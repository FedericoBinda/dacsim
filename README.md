dacsim
======

Simulation of the data acquisition chain of a detector


Build
-----

mkdir build

cd build

cmake ..

make

Run
---

cd build

./dacsim.py <input>


Output
------

A subdirectory output will be created.
The output files will be saved into this subdirectory.
